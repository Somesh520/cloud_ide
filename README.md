# Cloud IDE - Docker-based C++ Compiler

A sleek, cyberpunk-themed web-based C++ compiler that executes code safely using a Dockerized environment (for VPS) or direct execution with timeouts (for Render).

## Features
- **Real-time C++ Execution**: Compile and run C++ code instantly.
- **Multiple Test Cases**: Validate your code against multiple inputs.
- **Cyberpunk UI**: Modern, glitch-effect terminal interface.
- **Render Ready**: Optimized for free tier deployment on Render.com.

## How to Run Locally
1. Clone the repository.
2. Run `npm install`.
3. Start the server: `npm start`.
4. Open `index.html` in your browser.

## Deployment
This project is ready to be deployed on **Render** using the provided `Dockerfile`.
